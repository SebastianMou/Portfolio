Ensocio.mx: How We Built the Ultimate Freelancing Platform

Alright, here's the deal.

It's late, I'm still caffeinated, and I can't stop thinking about Ensocio.mx, the freelancing platform that we've just built. It's a brainchild born out of frustration and the sheer will to create something better than Fiverr or Upwork.

Features: What's Under the Hood?

Ensocio.mx is the Swiss Army knife of freelancing platforms. Check this out:

  • Home Page: Bam, you land on the home page, and the magic begins.
  • User Profiles: A place to show who you really are.
  • Messaging: Talk to anyone, about anything. Network like a pro.
  • Offers: Sellers post, buyers browse. Simple, right?
  • Posts: Create, edit, interact. Make your content count.
  • Admin Tools: The big guns that keep everything running smoothly.

URLs and Stuff

You'll find these bad boys scattered all over Ensocio.mx:

  • /: The gateway to our digital paradise.
  • /user_login/: Where you go to show you're in.
  • /profile/: Your digital identity, right here.
  • /inbox/: The hub for all your conversations.
  • /create_offer/<username>/: Craft your pitch, sell your skills.

What's Behind the Magic?

  • Django: Our backbone, the framework that does it all.
  • Python3: Because Python is just cool.
  • Bootstrap 5: Slick, modern, and responsive UI.
  • Stripe: Your payments, secure and seamless.
  • AJAX: Making things smooth and fast.

Getting It Running

For Linux

  1. git clone https://github.com/SebastianMou/EnsocioBetaV1.git
  2. cd EnsocioBetaV1
  3. python3 -m venv myenv
  4. source myenv/bin/activate
  5. pip install -r requirements.txt
  6. Run migrations and fire it up: 
    python3 manage.py makemigrations && python3 manage.py migrate
python3 manage.py runserver

For Windows

  1. git clone https://github.com/SebastianMou/EnsocioBetaV1.git
  2. cd EnsocioBetaV1
  3. python -m venv myenv
  4. myenv\Scripts\activate
  5. pip install -r requirements.txt
  6. Run migrations and fire it up: 
    python manage.py makemigrations && python manage.py migrate python manage.py runserver

Using Ensocio.mx

  1. Open your browser, type in localhost:8000.
  2. Sign up, log in, and explore. It's that simple.

Ensocio.mx: The Ultimate Freelancing Platform

Ensocio.mx is a powerful platform designed to connect freelancers and businesses. Inspired by market leaders like Fiverr and Upwork, we developed a system that offers comprehensive features including user profiles, messaging, offers, and secure payment integration. Whether you're a freelancer looking to showcase your skills or a business aiming to find the best talent, Ensocio.mx is here to bridge the gap.

With a modern, responsive design and advanced backend built using Django and Python, this platform ensures a smooth user experience. The platform's secure payment gateway, powered by Stripe, guarantees secure transactions for both freelancers and clients. Ensocio.mx is the one-stop solution for businesses seeking efficient, quality talent.
